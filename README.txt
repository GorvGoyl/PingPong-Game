------PING-PONG GAME-------

this is a single-user early 90's classic game.
this is a very popular game and one of the first games ever built.
this game is based on JAVA.



PROCEDURE:


OPEN cmd AND navigate to project directory in CMD
To COMPILE SOURCE CODE IN CMD: 
javac canvas.java


To RUN THE APPLICATION: 
java canvas
